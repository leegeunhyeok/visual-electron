# Visual Electron
[Electron](https://electronjs.org/) + [Vue.js](https://vuejs.org/) based data visualization tool

# Setup
Install dependencies modules and execute `dev` mode
```bash
yarn # or (npm install)
yarn run dev # or (npm run dev)
```
