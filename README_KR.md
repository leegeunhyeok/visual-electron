# Visual Electron
[Electron](https://electronjs.org/) + [Vue.js](https://vuejs.org/) 기반 데이터 시각화 도구

# 설정
의존성 모듈들을 설치한 후 `dev` 모드로 실행시킵니다.
```bash
yarn # 또는 (npm install)
yarn run dev # 또는 (npm run dev)
```
